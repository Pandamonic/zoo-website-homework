一、项目简介
本项目是一个基于动物园主题设计与开发的单页静态网站，本项目围绕小组前期提出的“动物 MBTI”创意展开设计，页面中部分视觉素材借助 AI 工具辅助生成，并由本人进行筛选、整理与统一风格处理。此外我还引入了动物园规则怪谈和一些网络热梗的元素。
网站通过大图主视觉、动物展示卡片、园区介绍、参观提示等模块，构成完整的浏览体验。

二、设计说明
网站以“一个普通的？动物园”为主题，整体风格兼顾自然氛围与现代设计感，突出动物展示与视觉表现。
本次修改将原本偏暗色、偏赛博的视觉风格调整为更接近现实动物园官网的暖色自然风格，主要使用米白、草绿、橙黄等色彩，让页面更像普通的动物园宣传网站。
在内容上仍保留 MBTI 动物拟人、规则怪谈与网络文化元素，使页面在常规展示之外带有一定的趣味性与反差感。
页面保留 CSS Checkbox Hack 的主题切换结构，在不使用跳转的前提下完成“表世界 / 里世界”的视觉变化。

三、页面结构
主要分为以下结构：
- 顶部导航区域
- 首页主视觉展示区
- 动物主题内容展示区
- 园区介绍/特色版块
- 参观提示或结尾信息区

四、JS 功能说明
本次作业新增 js 文件夹，每个功能单独放置一个 JS 文件：
1. nav-effect.js：实现顶部导航栏滚动缩放与当前浏览区域自动高亮。
2. image-magnifier.js：实现图片放大镜效果，鼠标移入动物图鉴或主视觉图片时显示局部放大区域。
3. cursor-paw.js：实现鼠标移动时的动物足迹效果，增强动物园主题氛围。

五、文件结构
- index.html：网页主体结构
- style.css：网页样式文件
- images/：网页图片素材
- js/：网页交互功能脚本
- JS功能说明.docx：本次 JavaScript 功能说明文档
