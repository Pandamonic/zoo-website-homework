一、项目简介
本项目是一个基于动物园主题设计与开发的单页静态网站，本项目围绕小组前期提出的“动物 MBTI”创意展开设计，页面中部分视觉素材借助 AI 工具辅助生成，并由本人进行筛选、整理与统一风格处理。，此外我还引入了动物园规则怪谈和一些网络热梗的元素。
网站通过大图主视觉、动物展示卡片、园区介绍、参观提示等模块，构成完整的浏览体验。

二、设计说明
   网站以“一个普通的？动物园”为主题，整体风格兼顾自然氛围与现代设计感，突出动物展示与视觉表现。
   页面整体采用深色基调，结合卡片式信息布局与分区展示结构，营造出略带神秘感的动物园视觉氛围。在内容上融合了 MBTI 动物拟人、规则怪谈与网络文化元素，使页面在常规展示之外带有一定的趣味性与反差感。
   页面通过 CSS Checkbox Hack 实现主题切换，在不使用跳转的前提下完成所谓的“表世界 / 里世界”的视觉变化。

三、 页面结构
   主要分为以下结构：
   - 顶部导航区域
   - 首页主视觉展示区
   - 动物主题内容展示区
   - 园区介绍/特色版块
   - 参观提示或结尾信息区
